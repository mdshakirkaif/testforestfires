Welcome to the FWI Prediction Project

This web application helps you predict the Fire Weather Index (FWI) using various meteorological and environmental parameters.
Enter your data, and get instant predictions to assist in wildfire risk assessment.
  • Features:
  • Easy-to-use form for entering weather and region data
  • Instant FWI prediction using advanced algorithms
  • Beautiful, responsive design for all devices
  • Secure and fast data processing
  • Parameters Used:
  • Temperature
  • Relative Humidity (RH)
  • Wind Speed (WS)
  • Rain
  • FFMC, DMC, ISI, Classes, Region
